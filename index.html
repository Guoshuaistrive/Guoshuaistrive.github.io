生成密钥

linux命令行生成密钥

  • 如果没有.ssh目录，自己创建一个

执行ssh-keygen -t rsa

一路按回车即可，生成密钥之后切换到.ssh目录

[root@guoshuai01 ~]# cd /root/.ssh/
[root@guoshuai01 .ssh]# ls

xshell生成密钥

在工具选项里面选择新建用户密钥生成向导,正常流程走下来就可以在磁盘目录下看到密钥

需要用到rz命令上传密钥到服务器；如果没有rz可以用yum install lrzsz安装

Wordpress部署验证

发表于 2017-12-14 | 分类于 exercise | 阅读次数:

Wordpress是一个开源的博客工具，用户可以在支持PHP和MySQL数据库的服务器上架设属于自己的网站，也可以把 WordPress当作一个内容管理系统（CMS）来使用。

  • 基于dockerfile构建mysql的镜像

  • 基于mysql镜像启动一个数据库容器，并进入容器登录数据库进行相关操作

        docker run -d           \
            -e MYSQL_USER=wordpress    \
            -e MYSQL_PASSWORD=word      \
            -e MYSQL_DATABASE=wp      \
            -e MYSQL_ROOT_PASSWORD=666666 \
            -v mysql:/var/lib/mysql --name mydb mysql:1

PS：进入容器后，数据库可以登录成功，wordpress数据库正常操作，但是创建其他数据库会出现ERROR
1045 (28000): Access denied for user 'username'@'IP' (using password: YES)错误。
