进入运行中的容器

[root@i-y61tbtqs docker-node-hello]# docker exec -t -i 43d44866883d /bin/bash
[root@43d44866883d /]#

执行这个命令之后会显示一个包含容器ID的命令提示符,表示我们已经进入这个运行中的容器。

  • 可以执行ps命令，查看容器中还运行着哪些进程。
[root@43d44866883d /]# ps -ef
