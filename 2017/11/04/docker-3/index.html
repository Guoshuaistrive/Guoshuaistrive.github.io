安装Docker问题：Cannot connect to the Docker daemon.

发表于 2017-11-04 | 分类于 analysis | 阅读次数:
  • Docker目前的学习还处于初级阶段，仅掌握了基础概念和基本命令执行而已，今天因为上海市举办的研究生论坛折腾了一天，白天没时间学习，晚上回到实验室想着在自己的Ubuntu上熟悉一下docker，没想到安装好之后一直显示Cannot connect to the Docker daemon问题，其实原因很简单还是自己没经验，经过几番调试终于显示成功。给大家分享一下过程，有遇到同样问题的同学可以参考，希望能有帮助！

是不是很水？？刚开始学习新的技术会有各种傻X的问题，慢慢排错吧。

自己笔记本的Linux是Ubuntu，与公司的Centos有区别，很用法多命令执行结果也有很大区别，对于刚刚接触docker的同学来说要多尝试多练习，先把docker的一些基础命令和掌握熟练，之后在容器里面安装软件或做项目会更块上手，避免不必要的错误。
