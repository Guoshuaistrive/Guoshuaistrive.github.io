数据库的安装以及远程访问

发表于 2018-02-26 | 分类于 exercise | 阅读次数:

首先安装mariadb

yum install mariadb mariadb-server

这时发现登录失败！看下进程

[root@guoshuai-01 ~]# ps -ef |grep mysql
root 3185 1568 0 18:46 pts/1 00:00:00 grep mysql
[root@guoshuai-01 ~]# service mariadb status
Redirecting to /bin/systemctl status mariadb.service
● mariadb.service - MariaDB database server
Loaded: loaded (/usr/lib/systemd/system/mariadb.service; disabled; vendor preset: disabled)
Active: inactive (dead)

启动数据库服务器

[root@guoshuai-01 ~]# service mariadb start
Redirecting to /bin/systemctl start mariadb.service

这时再看下进程

[root@guoshuai-01 ~]# ps -ef |grep mysql
mysql     3294     1  0 18:46 ?        00:00:00 /bin/sh /usr/bin/mysqld_safe --basedir=/usr
mysql     3456  3294  5 18:46 ?        00:00:00 /usr/libexec/mysqld --basedir=/usr --datadir=/var/lib/mysql --plugin-dir=/usr/lib64/mysql/plugin --log-error=/var/log/mariadb/mariadb.log --pid-file=/var/run/mariadb/mariadb.pid --socket=/var/lib/mysql/mysql.sock
root      3494  1568  0 18:46 pts/1    00:00:00 grep mysql

登陆成功！

数据库安全设置

/usr/bin/mysql_secure_installation

远程访问该数据库，需要再赋予权限

MariaDB [(none)]> use mysql
Reading table information for completion of table and column names
You can turn off this feature to get a quicker startup with -A

Database changed

  • 查看user,password,host这三个字段的权限分配情况

    MariaDB [mysql]>  select user,password,host from user;  

+------+-------------------------------------------+-------------+
| user | password                                  | host        |
+------+-------------------------------------------+-------------+
| root | *B2B366CA5C4697F31D4C55D61F0B17E70E5664EC | localhost   |
| root | *B2B366CA5C4697F31D4C55D61F0B17E70E5664EC | guoshuai-01 |
| root | *B2B366CA5C4697F31D4C55D61F0B17E70E5664EC | 127.0.0.1   |
| root | *B2B366CA5C4697F31D4C55D61F0B17E70E5664EC | ::1         |
+------+-------------------------------------------+-------------+
4 rows in set (0.00 sec)

    通过以上输出可以看出数据库默认只允许用户root在本地服务器（localhost）上登录，不允许其他主机远程连接

  • 为允许用户root使用密码在任何主机上连接该数据库，并赋予该用户所有权限

    MariaDB [mysql]> grant all privileges on *.* to root@"%" identified by "666666";
Query OK, 0 rows affected (0.01 sec)

MariaDB [mysql]> flush privileges;
Query OK, 0 rows affected (0.00 sec)

再查看user,password,host这三个字段的权限分配情况,会发现多了一行有“%”，这就意味着任何主机都被允许连接数据库
