Ubuntu16.04安装testlink

发表于 2018-03-04 | 分类于 learn | 阅读次数:

环境准备

1.Ubuntu16.04
2.Apache2
3.php7.0
4.mariadb10.0

上一篇博客在安装wordpress的时候已经安装好了php7.0和mariadb10.0的环境，此篇不再叙述

安装testlink

创建testlink数据库

CREATE DATABASE testlink;
CREATE USER 'testlinkuser'@'localhost' IDENTIFIED BY '444444';
GRANT ALL PRIVILEGES ON testlink.* TO 'testlinkuser'@'localhost' IDENTIFIED BY '444444' WITH GRANT OPTION;
FLUSH PRIVILEGES;
EXIT;

安装testlink

切换到/var/www/html目录下

下载压缩包wget http://github.com/TestLinkOpenSourceTRMS/testlink-code/archive/1.9.16.tar.gz

解压 tar -zxvf 1.9.16.tar.gz

将custom_config.inc.php.example重命名为custom_config.inc.php，并且替换文件内容

mv /var/www/html/testlink-code-1.9.16/custom_config.inc.php.example /var/www/html/testlink-code-1.9.16/custom_config.inc.php

// $tlCfg->log_path = '/var/testlink-ga-testlink-code/logs/'; /* unix example */
// $g_repositoryPath = '/var/testlink-ga-testlink-code/upload_area/';  /* unix example */

$tlCfg->log_path = '/var/www/html/testlink-code-1.9.16/logs/';
$g_repositoryPath = '/var/www/html/testlink-code-1.9.16/upload_area/';
$tlCfg->config_check_warning_mode = 'SILENT';

配置虚拟主机

cd /etc/apache2/sites-available/

cp 000-default.conf testlink.conf

vi testlink.conf

<VirtualHost 172.17.0.1>
ServerAdmin 767410530@qq.com
DocumentRoot /var/www/html/testlink-code-1.9.16/
ServerName www.gs.com
<Directory /var/www/html/testlink-code-1.9.16/>
Options FollowSymLinks
AllowOverride All
Order allow,deny
allow from all
</Directory>
ErrorLog /var/log/apache2/gs.com-error_log
CustomLog /var/log/apache2/gs.com-access_log common
</VirtualHost>

然后将该文件链接到sites-enabled文件（此处与centos配置不同）

ln -s /etc/apache2/sites-available/testlink.conf /etc/apache2/sites-enabled/

重启服务器

systemctl restart apache2

如果重启报错，要修改/etc/apache2/apache2.conf文件内容

# see README.Debian for details.

# Include generic snippets of statements
Include /etc/apache2/conf-enabled/*.conf

# Include the virtual host configurations:
Include /etc/apache2/sites-enabled/*.conf
# vim: syntax=apache ts=4 sw=4 sts=4 sr noet

大功告成！去浏览器访问ServerName即可看到testlink的页面

# testlink
