Centos7.4安装testlink

发表于 2018-03-07 | 分类于 exercise | 阅读次数:

这篇博客记录了在Centos7.4完全初始化的情况下从头到尾安装testlink的过程，其中包含centos上如何安装php7和mariadb10.1的过程，以及配置虚拟主机的步骤。

安装httpd

yum install httpd
systemctl start httpd
systemctl enable httpd

在本地curl localhost看看是否返回,然后再到浏览器用服务器ip测试

查看日志：/var/log/httpd/access_log error_log

安装php7

添加IUS的repo源，这是专门针对红帽系统提供rpm包更新的

yum install https://centos7.iuscommunity.org/ius-release.rpm

安装php7

yum install php70u php70u-pdo php70u-mysqlnd \
php70u-opcache php70u-xml php70u-gd \
php70u-devel php70u-json

查看结果

root@guoshuai-01 ~]# php -v
PHP 7.0.27 (cli) (built: Jan  4 2018 13:39:07) ( NTS )
Copyright (c) 1997-2017 The PHP Group
Zend Engine v3.0.0, Copyright (c) 1998-2017 Zend Technologies
with Zend OPcache v7.0.27, Copyright (c) 1999-2017, by Zend Technologies

测试 PHP 是否安装成功

cat <<EOF | sudo tee -a /var/www/html/phpinfo.php
<?php
phpinfo();
?>
EOF

重启httpd服务器 systemctl restart httpd

到浏览器http://ip/phpinfo.php测试

安装mariadb

数据库就是mariadb，红帽默认是5.5版本，testlink要求5.6以上的版本才能支持

cat <<EOF | sudo tee -a /etc/yum.repos.d/MariaDB.repo
# MariaDB 10.1 CentOS repository list - created 2017-01-14 03:11 UTC
# http://downloads.mariadb.org/mariadb/repositories/
[mariadb]
name = MariaDB
baseurl = http://yum.mariadb.org/10.1/centos7-amd64
gpgkey=https://yum.mariadb.org/RPM-GPG-KEY-MariaDB
gpgcheck=1 
EOF

安装mariadb 

yum install MariaDB-server MariaDB-client -y
systemctl start mariadb
systemctl enable mariadb

安全设置 /usr/bin/mysql_secure_installation

安装testlink

创建testlink数据库

CREATE DATABASE testlink;
CREATE USER 'testlinkuser'@'localhost' IDENTIFIED BY '20180307';
GRANT ALL PRIVILEGES ON testlink.* TO 'testlinkuser'@'localhost' IDENTIFIED BY '20180307' WITH GRANT OPTION;
FLUSH PRIVILEGES;

切换至/var/www/html目录,下载压缩包

wget http://github.com/TestLinkOpenSourceTRMS/testlink-code/archive/1.9.16.tar.gz

解压 tar -zxvf 1.9.16.tar.gz

创建一个custom文件

cp /var/www/html/testlink-code-1.9.16/custom_config.inc.php.example \
/var/www/html/testlink-code-1.9.16/custom_config.inc.php

打开该文件vi /var/www/html/testlink-code-1.9.16/custom_config.inc.php，配置如下

// $tlCfg->log_path = '/var/testlink-ga-testlink-code/logs/'; /* unix example */
// $g_repositoryPath = '/var/testlink-ga-testlink-code/upload_area/';  /* unix example */

$tlCfg->log_path = '/var/www/html/testlink-code-1.9.16/logs/';
$g_repositoryPath = '/var/www/html/testlink-code-1.9.16/upload_area/';
$tlCfg->config_check_warning_mode = 'SILENT';

设置权限 chown -R apache:apache /var/www/html/testlink-code-1.9.16

此时，可以先用ip访问判断一下testlink是否联通

配置虚拟主机

cat <<EOF | sudo tee -a /etc/httpd/conf.d/testlink.conf    
<VirtualHost *:80>    
ServerAdmin admin@gs.com    
DocumentRoot /var/www/html/testlink-code-1.9.16/    
ServerName testlink.gs.com    
ServerAlias www.testlink.gs.com    
<Directory /var/www/html/testlink-code-1.9.16/>    
Options FollowSymLinks    
AllowOverride All    
Order allow,deny    
allow from all    
</Directory>    
ErrorLog /var/log/httpd/testlink.gs.com-error_log    
CustomLog /var/log/httpd/testlink.gs.com-access_log common    
</VirtualHost>    
EOF

重启服务器 systemctl restart httpd

在C:\windows\system32\drivers\etc\hosts文件中加入域名到本地IP地址的映射

150.129.193.14 testlink.gs.com

结束安装

一路点击continue即可，进入数据库页面之后的配置如下

Database Type: MySQL/MariaDB (5.6+ / 10.+)
Database host: localhost
Database name: testlink
Table prefix: <LEAVE IT EMPTY>
Database admin login: root
Database admin password: <your-MariaDB-root-password>
TestLink DB login: testlinkuser
TestLink DB password: yourpassword

现在即可登录网页使用testlink，登录名称和密码均为admin

Login Name: admin
Password: admin

登录之后即可修改你的密码

参考文章 https://www.vultr.com/docs/how-to-install-testlink-on-centos-7
