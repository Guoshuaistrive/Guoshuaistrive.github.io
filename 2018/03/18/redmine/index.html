Ubuntu16.04安装redmine的过程

发表于 2018-03-18 | 分类于 exercise | 阅读次数:

更改系统默认源

Ubuntu默认使用的国外的源，在更新的时候会很慢。中国科技大学的源一般使用起来比较快，直接选择中科大的源。

先备份 sudo cp /etc/apt/sources.list /etc/apt/sources.list.bak

删除默认源，添加中科大的源sudo vi /etc/apt/sources.list

# USTC (University of Science and Technology of Thina)
# xenial is ubuntu 16.04 version
deb http://mirrors.ustc.edu.cn/ubuntu/ xenial main restricted universe multiverse
deb http://mirrors.ustc.edu.cn/ubuntu/ xenial-security main restricted universe multiverse
deb http://mirrors.ustc.edu.cn/ubuntu/ xenial-updates main restricted universe multiverse
deb http://mirrors.ustc.edu.cn/ubuntu/ xenial-proposed main restricted universe multiverse
deb http://mirrors.ustc.edu.cn/ubuntu/ xenial-backports main restricted universe multiverse
deb-src http://mirrors.ustc.edu.cn/ubuntu/ xenial main restricted universe multiverse
deb-src http://mirrors.ustc.edu.cn/ubuntu/ xenial-security main restricted universe multiverse
deb-src http://mirrors.ustc.edu.cn/ubuntu/ xenial-updates main restricted universe multiverse
deb-src http://mirrors.ustc.edu.cn/ubuntu/ xenial-proposed main restricted universe multiverse
deb-src http://mirrors.ustc.edu.cn/ubuntu/ xenial-backports main restricted universe multiverse

Ubuntu 16.04的版本代号为xenial，所以上面全部是xenial，如果是其他版本的Ubuntu，也是同样的道理，只不过修改代号就可以了。

更新列表 sudo apt-get update

然后再apt-get upgrade一下

安装依赖

apt-get install mysql-server mysql-client libmysqlclient-dev imagemagick libmagickwand-dev libssl-dev libcurl4-openssl-dev git-core subversion

安装rvm和ruby

gpg --keyserver hkp://keys.gnupg.net --recv-keys D39DC0E3

curl -L https://get.rvm.io | bash -s stable --ruby=2.2.5

  • 如果安装出现错误，按照人家提示的信息再运行一条命令即可。

重载RVM，并添加到 .bashrc

source /usr/local/rvm/scripts/rvm

echo '[[ -s "/usr/local/rvm/scripts/rvm" ]] && source "/usr/local/rvm/scripts/rvm"' >> ~/.bashrc

配置Redmine数据库

create database redmine;
create user redmine@localhost identified by '20180318';
grant all privileges on redmine.* to redmine@localhost identified by '20180318';
flush privileges;

安装Phusion Passenger和Nginx

gem install passenger --no-ri --no-rdoc

passenger-install-nginx-module

这条命令运行时会有提示：默认选择Ruby and Python； 之后选择1.Yes：download…..; 往下enter即可

配置Nginx

cd /opt/nginx/conf/

打开文件vi nginx.conf，在末尾加一行

include vhost/*.conf;

创建redmine虚拟主机配置文件

新建目录 mkdir -p /opt/nginx/conf/vhost

切换到该目录下cd /opt/nginx/conf/vhost/

创建配置文件vi redmine.conf

server {
        listen       80;
        server_name  www.redmine.me;

        root /var/www/redmine/public;
        passenger_enabled on;
        client_max_body_size      10m; # Max attachemnt size

        # redirect server error pages to the static page /50x.html
        #
        error_page   500 502 503 504  /50x.html;
        location = /50x.html {
            root   html;
        }
    }

配置Nginx系统启动

切换到系统目录 cd /lib/systemd/system/
创建文件 vi nginx.service

[Unit]
Description=The NGINX HTTP and reverse proxy server
After=syslog.target network.target remote-fs.target nss-lookup.target

[Service]
Type=forking
PIDFile=/opt/nginx/logs/nginx.pid
ExecStartPre=/opt/nginx/sbin/nginx -t
ExecStart=/opt/nginx/sbin/nginx
ExecReload=/bin/kill -s HUP $MAINPID
ExecStop=/bin/kill -s QUIT $MAINPID
PrivateTmp=true

[Install]
WantedBy=multi-user.target

重载系统服务systemctl daemon-reload

系统命令启动Nginx systemctl start nginx

检查Nginx netstat -plntu | grep nginx

安装Redmine

创建redmine目录mkdir -p /var/www/

切换到该目录下 cd /var/www/

下载redmine svn co https://svn.redmine.org/redmine/branches/3.2-stable redmine

进入redmine目录，复制配置文件

cd redmine
cp config/configuration.yml.example config/configuration.yml
cp config/database.yml.example config/database.yml

编辑数据库文件,修改production的配置vi config/database.yml

production:
  adapter: mysql2
  database: redmine
  host: localhost
  username: redmine
  password: "20180318"
  encoding: utf8

在redmine目录下，创建一个新目录改变一下owner

mkdir -p tmp tmp/pdf public/plugin_assets
sudo chown -R www-data:www-data files log tmp public/plugin_assets
sudo chmod -R 775 files log tmp public/plugin_assets

为Redmine安装bundler和gem依赖

gem install bundler
bundle install --without development test

Now generate the secret token and then generate the database:
bundle exec rake generate_secret_token
RAILS_ENV=production bundle exec rake db:migrate
RAILS_ENV=production bundle exec rake redmine:load_default_data

重启Nginx systemctl restart nginx，在网页访问redmine的域名

# redmine
